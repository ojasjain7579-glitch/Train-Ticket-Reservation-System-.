# Train Ticket Reservation System

## Oasis Infobyte Internship Project

This project was developed as part of the **Oasis Infobyte Internship Program**.

## Overview

Train Ticket Reservation System is a Java-based web application that enables users to search trains, check seat availability, book tickets, and manage reservations efficiently.

## Features

* User Registration and Login
* Train Search
* Seat Availability Check
* Ticket Booking
* Fare Enquiry
* Booking History
* Admin Dashboard
* Train Management System

## Technologies Used

* Java
* Servlets
* JDBC
* Oracle Database
* HTML
* CSS

## Project Setup

1. Install Java JDK
2. Configure Oracle Database
3. Install Apache Tomcat Server
4. Import the project into Eclipse IDE
5. Deploy and run on Tomcat
