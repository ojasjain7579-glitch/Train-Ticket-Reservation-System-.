## OIBSIP Java Development Internship Projects
This repository contains projects completed during Oasis Infobyte Internship.

## Task - 1 Online reservation system

## Features

* User Registration and Login
* Train Search
* Seat Availability Check
* Ticket Booking
* Fare Enquiry
* Booking History
* Admin Dashboard
* Train Management System

## Technologies Used

* Java
* Servlets
* JDBC
* Oracle Database
* HTML
* CSS

## Project Setup

1. Install Java JDK
2. Configure Oracle Database
3. Install Apache Tomcat Server
4. Import the project into Eclipse IDE
5. Deploy and run on Tomcat
